# 📊 HR Employee Attrition Analytics

An end-to-end **HR Analytics** project built using **Microsoft Excel, Power Query, DAX, Power BI, Canva, and Zapier** to transform employee data into actionable business insights. The project focuses on workforce trends, employee attrition, employee engagement, burnout, promotion analysis, and workflow automation through interactive dashboards.


## 🛠️ Tools & Technologies

- Microsoft Excel
- Power BI
- Power Query
- DAX
- Canva
- Zapier


## 🔄 Project Workflow

### 📌 1. Data Preparation (Microsoft Excel)

- Cleaned the HR dataset by handling missing values, correcting inconsistencies, and validating records.
- Standardized the dataset to ensure accuracy and consistency before analysis.

### 📌 2. Data Transformation (Power Query)

- Performed data cleaning and transformation using Power Query.
- Created calculated columns and prepared the dataset for reporting.

### 📌 3. Data Modeling & DAX (Power BI)

- Developed custom DAX measures and KPIs for workforce and attrition analysis.
- Built an optimized data model to support interactive reporting.

### 📌 4. Dashboard Development (Power BI)

Designed two interactive dashboards:

- **Workforce Intelligence Dashboard**
- **Employee Attrition Dashboard**

Implemented KPI cards, slicers, AI visuals, and interactive charts to deliver actionable HR insights.

### 📌 5. HR Campaign Poster (Canva)

- Designed a professional **Zoho-inspired HR culture poster** highlighting employee-first values, workplace culture, and key HR initiatives.

### 📌 6. Workflow Automation (Zapier)

- Automated the project workflow by integrating **Canva, Google Drive, and Gmail** using Zapier, enabling seamless file storage and automated email notifications.


## 📊 Dashboard Highlights

### 👥 Workforce Intelligence Dashboard

- Monitors workforce distribution across departments and job roles.
- Analyzes employee demographics, education levels, and work modes.
- Tracks employee count, average salary, engagement score, work-life balance, burnout score, and AI adoption.
- Provides interactive insights into workforce composition and organizational performance.

### 📉 Employee Attrition Dashboard

- Identifies attrition trends across departments, age groups, gender, and salary categories.
- Evaluates the impact of promotion waiting time and burnout on employee retention.
- Enables root cause analysis using the Decomposition Tree and interactive filtering.
- Supports HR decision-making through department-wise attrition comparisons and workforce risk analysis.


## 📈 Key Metrics

### 👥 Workforce Intelligence Dashboard

- Total Employees
- Average Monthly Salary
- Average Engagement Score
- Average Burnout Score
- Average Work-Life Balance Score
- AI Adoption Rate

### 📉 Employee Attrition Dashboard

- Employees Left
- Attrition Rate
- Department-wise Attrition
- Attrition by Age Group
- Attrition by Salary Category
- Promotion Waiting Analysis


## 💡 Key Insights

- Engineering has the largest workforce and records the highest employee attrition.
- Employees aged **26–45 years** represent the highest proportion of the workforce and employee exits.
- Hybrid is the most preferred work mode across the organization.
- Higher burnout levels and longer promotion waiting periods are associated with increased attrition.
- Sales department reports the highest average burnout score, highlighting potential employee well-being concerns.
- Employee engagement and work-life balance emerge as key factors influencing workforce retention.
